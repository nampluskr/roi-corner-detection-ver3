# roi-corner-detection-ver3

이 프로젝트는 [260712_roi-corner-detection-ver2](../260712_roi-corner-detection-ver2/)의 폴더/파일
구성을 재구성한 결과를 확립하는 작업 공간이다.

## 현재 상태

ver2는 [model-assembly.md](../260712_roi-corner-detection-ver2/docs/architecture/model-assembly.md)를
canonical SSOT로 하는 구조를 갖고 있다. 이 문서와 ver2의 다른 SSOT/canonical 문서는 재구성 작업의
**참고 자료**로만 사용하며, ver3 자체의 구속력 있는 기준이 되지 않는다.

ver3에서 사용자의 재구성 요청을 실제로 반영하고, 그 결과를 ver3의 새 canonical 문서로 확립한다.
확정된 재구성 방안과 그 근거는 `docs/`에 기록한다.

## 재구성 대상

- 참고 경로: [260712_roi-corner-detection-ver2](../260712_roi-corner-detection-ver2/)
- 참고 SSOT 문서: [docs/architecture/model-assembly.md](../260712_roi-corner-detection-ver2/docs/architecture/model-assembly.md)

문서 색인과 작업 절차는 [docs/README.md](docs/README.md)를 따른다.
